CSS

  • Enter your CSS notes here
  • CSS is a programming language used by developers to define how webpage content is presented to users.
  • Whereas HTML strictly handles the structure and content of a webpage, CSS defines the colors, aesthetics, visual layout, and organization. We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.

Git

  • Enter your Git notes here
  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Using Git commands, we can initiate (or create) a repo, add our code to the repo, and view the different versions of our code, which are each stored in a unique commit.
  • Git is great for tracking changes locally on your own computer, but to collaborate with others, you need to use another tool, GitHub. GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories that are stored remotely. Using GitHub not only allows us to store our code, but also to easily collaborate with others.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
